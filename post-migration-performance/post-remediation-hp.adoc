== Après la migration - FLI en ligne

Après avoir supprimé l'importation LUN, vérifiez si la LUN est configurée avec  `-space-reserve disabled` et  `-space-allocation enabled`. Si ce n'est pas le cas, modifiez-la en conséquence.

Pour que la fonction SCSI UNMAP fonctionne correctement, veuillez consulter la documentation du fournisseur afin de vérifier les versions logicielles compatibles. La combinaison Veritas VxVM + VxFS est requise pour activer cette fonction.

.Étapes
. Identifier les périphériques de stockage.
+
[source, cli]
----
ioscan -fnC disk
insf -e
sanlun lun show all
sanlun lun show all -pv
sanlun lun show -wwpn
----
. Importer VG et mettre le VG en ligne.
+
[source, cli]
----
vgimport /dev/vg_name /dev/disk/disk_name
vgchange -a y /dev/vg_name
----
. Identifiez le groupe de volumes et les systèmes de fichiers.
+
[source, cli]
----
vgdisplay -v /dev/vg_name
vgdisplay /dev/vg_name
lvdisplay /dev/vg_name/lv_name
----
. Montez le système de fichiers.
+
[source, cli]
----
bdf
mount /dev/vg_name/vol_name /mount_point
----
