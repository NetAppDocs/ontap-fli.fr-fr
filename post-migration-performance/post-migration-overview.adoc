= En savoir plus sur la post-migration ONTAP FLI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Cette section décrit les opérations à effectuer une fois la migration Foreign LUN Import (FLI) terminée. Il existe des étapes de nettoyage générales, des étapes de validation des performances et des étapes pour chaque type d’hôte (AIX, ESXi, HP, Linux, Solaris et Windows).

Les étapes post-migration sont importantes pour garantir que l'environnement est laissé dans un état propre et maintenable, et pour valider que la migration a été réussie du point de vue de l'intégrité des données et des performances.
