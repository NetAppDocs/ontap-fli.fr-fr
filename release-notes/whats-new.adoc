Découvrez les nouvelles fonctionnalités disponibles dans ONTAP pour la migration des données SAN.



== ONTAP 9.19.1

* À partir d'ONTAP 9.19.1, ONTAP FLI prend en charge le protocole iSCSI pour la connectivité du backend.




== ONTAP 9.17.1

* À partir de ONTAP 9.17.1, les systèmes ASA R2 sont pris en charge pour la migration FLI.
+
En savoir plus sur link:https://docs.netapp.com/us-en/asa-r2/get-started/learn-about.html["Systèmes ASA R2"^] .
